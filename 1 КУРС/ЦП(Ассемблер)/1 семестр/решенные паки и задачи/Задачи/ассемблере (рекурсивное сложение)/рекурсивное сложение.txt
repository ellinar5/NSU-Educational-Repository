Подается на вход некая n. Напрмер - 3

Проверка
1. n =< 0 переводит к halt, выводит в результате 0 и заканчивает программу
2. n > 0 переводит в метку и делает:

recurse:
*1. Записать в стек текущее число
*2. Извлекаем его в r1 для использования в рекурсии
*3. Декрементируем (n-1)
*4. Прибавляем n к резельтату
*5. Сравниваем n с чем-то в r2
*6. Если n == 0, переходим к return
*7. Сохраняем результат на стеке
*8. Рекурсивная функция

return:
**1. Извлекаем значение из стека
**2. В result загружаем значение из стека 

выводим резельтат

По примеру:

n = 3 => больше 0 => проводим опирации

f(3) = 3 + f(2) = 3 + 3 = 6
f(2) = 2 + f(1) = 2 + 1 =3
f(1) = 1 + f(0) = 1 + 0 = 1
f(0) = 0 

вывод 6
