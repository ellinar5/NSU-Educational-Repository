map — это функция  которая принимает функцию и список, и применяет эту функцию к каждому элементу списка, возвращая новый список с результатами

tail - функция для получения всех элементов списка, кроме первого

last - функция возвращает последний элемент списка

init - функция возвращает все элементы списка, кроме последнего

concat - функция для объединения списка списков в один список

take - функция для извлечения первых n элементов из списка

nub - функция для удаления дубликатов из списка, сохраняя при этом порядок элементов

union - функция для объединения двух списков, удаляя дубликаты

intersect - функция для нахождения пересечения двух списков

Оператор \ удаляет из первого списка все элементы, которые присутствуют во втором списке. При этом учитывается количество вхождений элементов

scanl  - это фукция для создания списка, который содержит накопленные результаты применения заданной бинарной функции к элементам списка, начиная с начального значения

null - это функция которая проверяет, есть ли элементы в списке

product - это функция для перемножения всех элементов списка

permutations - это функция для генерации всех возможных перестановок элементов списка

minBound - минимальное значение для конкретного класса типов

filter — это функция, которая принимает предикат (функцию, возвращающую True или False) и список, и возвращает новый список, содержащий только те элементы, для которых предикат возвращает True

zipWith — это функция, которая принимает бинарную функцию и два списка, и применяет эту функцию к соответствующим элементам обоих списков, создавая новый список.

61. Функция t61 возвращает Just с результатом деления, если деление возможно, или Nothing, если второе число равно нулю.

Если вызвать t61 10 2, результат будет Just 5, так как 10 делится на 2 без остатка.
Если вызвать t61 10 0, результат будет Nothing, так как деление на ноль не определено.
Если вызвать t61 10 3, результат будет Just 3, так как 10 делится на 3 с целой частью 3.

62. Функция t62 возвращает элемент списка по указанному индексу, если индекс допустим, или Nothing`, если индекс недопустим. 

Если вызвать t62 [1, 2, 3, 4] 2, результат будет Just 3, так как элемент на индексе 2 в списке — это 3.
Если вызвать t62 [1, 2, 3, 4] 4, результат будет Nothing, так как индекс 4 выходит за пределы списка.
Если вызвать t62 [1, 2, 3, 4] (-1), результат также будет Nothing, так как индекс отрицательный
