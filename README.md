# NSU-Educational-Repository
В данном репозитории размещены учебные проекты и выполненные задачи за годы обучения. Здесь представлены задания на языке C, материалы по работе с операционными системами Solaris и Linux, а также задачи и лабораторные работы на языках Python и C++. Репозиторий отражает процесс обучения, практику программирования и развитие навыков работы с кодом...
